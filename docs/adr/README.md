# Architecture Decision Records

This directory contains ADRs (Architecture Decision Records) for the Fundy project.

ADRs capture significant architectural decisions, the context that led to them, and the consequences. They are numbered sequentially and immutable — superseded records are marked rather than deleted.

## Status Legend

| Status       | Meaning                  |
| ------------ | ------------------------ |
| `proposed`   | Under discussion         |
| `accepted`   | Decision made and active |
| `deprecated` | No longer relevant       |
| `superseded` | Replaced by a later ADR  |
